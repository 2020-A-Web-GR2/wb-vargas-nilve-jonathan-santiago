NestJS is a framework for 
building efficient, scalable Node.js web applications.

## instalar nestjs
```
$ npm i -g @nestjs/cli
```

## crear nuevo proyecto
```
$ nest new project-name

```
## correr un proyecto
```
$ npm run start:dev
```
